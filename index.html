What is this page? A demo page for a library of reusable custom elements (html tags) using Web Components API written in vanilla JavaScript.

Ok, But Why? I wanted to learn hopw to make my own components, spread the word about Web Components as a technology and hopefully someone can use them for things...

So what the heck are they? Accoring to MDN Web Components are:

Web Components is a suite of different technologies allowing you to create reusable custom elements — with their functionality encapsulated away from the rest of your code — and utilize them in your web apps.

And here is more:

Custom elements are treated by the browser as regular html tags. Shadow DOM lets you create a "shadow" DOM tree on any element; hidden away to avoid collision with other parts of the page. HTML templates and slots allow to create markup structures that can be used over and over.

Find more information about Web Components on MDN here

Ok, fine. How do I use them? See below. Also feel free to open your browser dev tools to poke around.

Navbar component

A simple navbar component that has multiple links and a logo.

<nav-bar></nav-bar>

Example code

<nav-bar> <img src='img/logo.png'> <a href="https://www.somepage.com/">Link1</a> <a href="https://www.somepage.com/">Link2</a> <a href="https://www.somepage.com/">Link3</a> <a href="https://www.somepage.com/">Link4</a> <a href="https://www.somepage.com/">Link5</a> </nav-bar>

Examples of navbar with and without logo.

Button Component

Types: outlined, regular, positive, negative, and disabled. Attributes: label and type.

<my-button></my-button>

Example code

<my-button label='Outlined' type="outlined" onClick=displayAlert()></my-button>
"Reveal-text" component

Reveals text on click and then removes itself from the DOM. It can take any number of paragraphs. You can apply custom styles to each of the paragraphs.

<reveal-text></reveal-text>

Example code

<reveal-text> <p class="your-custom-style">some text</p> <p>some mopre text</p> <p>even more text</p> </reveal-text>

Tooltip component

Displays text on hover. Attributes: img, text, size.

<tooltip-component></tooltip-component>

Example code

<tooltip-component img="img.png" text="text to display" size="large"> </tooltip-component>
Image-Card component

Displays an image and a description. Attributes: image, text, size. The text should be short 2-3 sentence paragraph.

<image-card></image-card>

Example code

<image-card img="img/cat1.jpg" size="small" text="Some text"></image-card>
Modal Component

Displays a box/popup window on click. Attributes: label, text.

<modal-button></modal-button>

Example code

<modal-button label='Open Modal' text="Some text"></modal-button>
