Small library of reusable custom elements using

Web components
  • set of technologies allowing to create reusable custom elements
  • their functionality encapsulated away from the rest of the code
  • do not depend on any external libraries/frameworks
  • easy to use across different web apps

Three main technologies:

  • Custom elements: custom elements treated by the browser as regular html tags.
  • Shadow DOM: lets create a "shadow" DOM tree on any element; hidden away to avoid collision with other parts of the document.
  • HTML templates: The <template> and <slot> elements allow to write markup templates that are not displayed on the page.
